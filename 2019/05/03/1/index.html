一人の体験2.0（一）——大学生穷游日本2019

发表于 2019-05-03 更新于 2020-02-03 分类于 旅行 ， 日本
本文字数： 2.1k

前言

大家好我是小K同学，大学一年生一枚，喜欢旅行。
本次图文的内容是我寒假去日本时候的精彩分享。
为什么标题是大学生穷游日本2.0呢？因为这是我第二次一个人日本自由行啦。
第一次是在我高一暑假的时候，我妈叫我自己去，从此爱上日本，所以很多地方都比较有经验，我觉得这次的行程规划是最好的一次了👌比如交通，日本的住宿等等，下面会提及到。本图文不会有很多图片，每个景点大概两三张左右，想看更多还请酷基们自己去探索呢！其实是因为很多图片都放在电脑里懒得拿出来了

行程准备

  • 首先po出我本次的行程，有兴趣冬天去日本的酷基可以参考下我的行程，这个行程包括了冬天日本热门的地点，而且各个地方的距离也不是太远(除了高山到东京，高速巴士耗时5个小时)

  • 准备：日本对外国旅客是有很多优惠的政策的。根据我的行程你会看到有一个升龙道巴士券的东东，其实它就是一张高速巴士的套票哦，我买的是三日券(某宝有售)，430软妹币你就可以在升龙道地区随意乘坐高速巴士，真的很优惠呢，它帮我在升龙道地区节省了一半的路费。

  • 东京的话我是打算待三天的，上某宝搜索了下发现有东京地铁三日券,果断入手，东京有十三条地铁线，坐地铁就可以畅游东京了哦。当然我还有张西瓜卡，之前去日本的时候买的(日本的储值卡，可以在便利店买东西，坐公交，坐地铁，坐电车)

日本的铁道交通很发达，东京地铁线路图如下，基本上没有什么地方是搭地铁去不到的。

本次行程一共10天。

机票方面我是提前了一个月订，在飞猪订的，名古屋进东京出，一共花费1800RMB，这算是比较便宜的廉价机票了，一般廉价机票是没有托运的。不过我一个人很简单，就只带一个背包(只是背包很大而已)，问题不大！

搭机时间我是喜欢坐凌晨机，一到日本马上就可以去玩
真是能玩的时间一点都不耽误。但是这次凌晨没有廉价机票卖，有点可惜。

行程开始

————>废话少说，马上起飞吧！！！

あと２０分が着陸します。まもなく日本です。
みんな様、名古屋へようこそ!
欢迎来到日本名古屋市哦
着陆后弄完手续大概都下午5点钟了，马上坐电车出机场去找吃的。

很多人不知道日本的电车究竟是怎样的，在这里解答一下。日本的电车其实就是行走在地面或高架上的轨道交通，日本的地铁和电车车型其实是一样的，电车的话分为普通，急行，特急三种速度。我们可以理解为电车是地面走上的地铁。后面会讲到我电车与地铁无缝切换时的尴尬经历。

好香！

一下电车就找到了家一蘭拉面，点了一碗招牌的一蘭拉面+两个鸡蛋，承惠120RMB，一下飞机就大出血。

  • 这家店很小但是很精致，在门口的精算机自行投币点餐后出来一张小纸条，然后你拿着这张小纸条去找一个卡座坐下把纸条递给帘子里的服务员就好了，帘子送餐后会关起来，无论你吃相多难看都不用担心。坐在里面吃很有安全感(感叹日本人想得真是周到)。需要服务按一下卡座里的铃就好啦。

  • 再加一点，日本人喜欢喝冰水，无论什么季节。我冬天去照样喝冰水，明明外面下雪了，他还是给你冰水，也有用茶的，原因就是茶和冰水不同，一个有茶叶，一个有冰块，比较上来就显得用心多了，这也是历史原因，直到至今“接待贵宾要用冰”的概念也根深蒂固。

晚上就出去街上逛逛，名古屋不愧是大城市呢。

第二天就是在名古屋逛逛啦😄

根据前一晚上睡前做的功课，发现名古屋地铁的名城线可以经过名古屋的大部分热门景点哦。实在太方便了！启程！！！

第一站：名古屋城

日本对于历史文物的保护还是很不错的。
名古屋城是德川家康于1612年所建，经二战战火损毁，1959年修复了幸存的建筑。虽已不复当年旧貌，但城堡的规模仍可一窥，加之其内的2000多株樱花树，花开时节所有建筑掩映于粉烟之中，景象动人。
(引用自百度百科)

名古屋城并不只是我拍照的这几张，它像北京故宫一样，有城墙，有建筑，有庭院，但相比于故宫当然还是小很多的啦

进去几年后我们可以发现虽然经过了战火损毁，但是修复程度很高，修复也很精致，基本上是和历史上的名古屋城一模一样。但是日本很多历史古迹(给人参观的)都只是一个外壳，建筑内其实是没有任何东西的(比如家具之类的)，只有一个指示牌说明这里之前是放什么东西的，这应该是日本政府防范文物被破坏的一个措施吧

第二站：热田神宫

热田神宫位于日本爱知县名古屋市的中心地带，是日本三大神社之一，为日本皇室庄严神圣之文物。面积为19万平米，采用神明建筑风格建造，屋脊用铜板覆盖。
热田神宫主祭神为热田大神（天丛云剑），相殿祭祀天照大神、素盏明尊、日本武尊等。(源自百度百科)

日本的宫不用说和我们中国一样是祭祀用的，神宫则祭祀神灵。具体的我就不太清楚，这要考究到日本的历史文化了。

里面最吸引我的还是许愿墙了⭐️上面各种各样的祝福给予着我们希望，其实我们世界上不管哪个国家，哪个地区，谁不希望大家过得幸福，天天开心呢？
