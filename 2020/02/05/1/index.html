日本语学习笔记——标日初第15课

发表于 2020-02-05 分类于 学习笔记 ， 日本语 ， 标准日本语 ， 第15课
本文字数： 852

语法

😭

1. 動詞て　います
  • 表示正在进行，我们英语所说的ing形式
    组成：动词て型＋います
    e.g.
    我正在图书馆看书 ——> わたしは図書館に本を読んでいます。
  • 表示（过去）正在进行的时候，可以把后面的「います」改成「いました」
    组成：动词て型＋いました
    e.g.
    我昨天白天在图书馆看书 ——> 昨日の昼わたしは図書館に本を読んでいました。
2. 動詞ても　いいです

  • 表示许可，可以的
    组成：动词て型＋も＋いいです
    e.g.
    我今天可以休息吗 ——> きょう私は休んでもいいですか？
    在家工作也可以 ——> 家で仕事をしてもいいです。
    答：可以的 ——> してもいいです。（いいです。）

  • 「動詞ても かまいません」 也可以表示许可
    组成与上面相同
    かまいます————>妨碍
    かまいません———>不妨碍
    e.g.
    在家工作也不妨碍（可以） ——> 家で仕事をしてもかまいません。
    答：可以的 ——> してもかまいません（いいです。）

3. 動詞ては いけません

  • 表示禁止，不行
    いけます——>可行
    组成：动词て型＋は＋いけません
    e.g.
    银行前禁止停车 ——> 銀行の前に駐車してはいけません。

  • 「だめです」也可以表示禁止
    组成：与上面相同
    e.g.
    银行前禁止停车 ——> 銀行の前に駐車してはだめです。

4．名詞[附着的地方] に 動詞
  • 表示人或物体的附着点，例如人或物体停留在交通工具上或椅子上时
    e.g.
    小李上了电车 ——> 李さんは電車に乗りました。
    请坐在这把椅子上 ——> このいすに座ってください。
    小李把文件放进了包里 ——> 李さんはかばんに書類を入れました。
5. 名詞[目的地] に 動詞
  • 表示移动行为的目的地时可以用「へ」也可以用「に」
    e.g.
    小李去医院 ——> 李さんは病院に（へ）行きます。
    小李回家了 ——> 李さんは家に（へ）帰りました。

总结

😊

  1. 要分清楚「てもいいです」和「てはいけません」，表示允许时用「ても」，禁止时用「ては」，不要把「て」错写成「で」！！！

  2. 对于「に」的用法要经常复习，熟记。
